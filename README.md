# EcoPanel - Sitio Web Corporativo

EcoPanel es un proyecto académico desarrollado en equipo para la materia **Plataformas Abiertas II**.  
El proyecto consiste en la creación de un sitio web corporativo para una empresa ficticia dedicada a la instalación y comercialización de paneles solares.

## Descripción

El objetivo del proyecto fue diseñar y estructurar un sitio web empresarial que permita presentar información institucional, servicios y contenido informativo relacionado con energía solar.

Se trabajó en la organización del contenido, estructura de navegación y desarrollo de múltiples secciones simulando un entorno real de negocio.

## Tecnologías Utilizadas

- HTML5  
- CSS3  
- GitHub  
- Visual Studio Code  

## Requisitos Previos

- Navegador web moderno (Chrome, Edge, Firefox, etc.)

## Instalación

1. Clona este repositorio:

   git clone [https://github.com/tu-usuario/ecopanel.git](https://github.com/Chamo1507/EcoPanel.git)

2. Ingresa a la carpeta del proyecto:

   cd ecopanel

3. Abre el archivo `index.html` en tu navegador.

## Uso

El sitio permite navegar entre las siguientes secciones:

- Index  
- Nosotros  
- Servicios (Venta y Compra)  
- Blog  
- Contacto  

Cada sección presenta contenido estructurado como en una página corporativa real.

## Características

- Sitio web corporativo estructurado en múltiples páginas
- Navegación entre secciones
- Organización semántica del contenido
- Proyecto desarrollado en equipo
